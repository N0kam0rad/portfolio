How I built The Helix Class

A vertical-scroll webtoon world set inside a luxury biotech academy where the mirrors do not behave.

The spark

I wanted a polished story world with fashion, class tension, biotech weirdness, and a visual language strong enough to hold a webtoon.

How I built it

I started with character references, visual rules, and a world premise before moving into episode structure. AI helped generate visual anchors, but the story logic needed human taste and guardrails.

What is next
