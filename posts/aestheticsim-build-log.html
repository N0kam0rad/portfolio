The problem

Medspa consultations are expensive for the client and high-pressure for the clinician. Most of the time, the client leaves with a verbal explanation they can barely remember and no visual proof of what was discussed. The clinician has to rebuild the same conversation from scratch every time. That gap was the spark.

AestheticSim is a two-sided workflow: the client fills out an intake form, the clinician reviews a structured analysis board, and the session ends with a downloadable PDF consultation board the client takes home.

Key flows built

  • Role selection landing: clinic, client, or admin — each enters the same system differently
  • Client intake wizard: structured questions, privacy-safe mock data
  • Booked client gate: confirmation screen before intake unlocks
  • Clinician review board: client list, analysis states, facial mapping
  • Admin panel: branding, booking, disclaimer, integrations
  • Downloadable consultation board: executive summary, aging analysis, skin quality, treatment phases, cost forecast
The strongest demo path: open Marcus Webb → run through the clinician board → click "Generate board" → download the PDF. That is the one sequence that shows the full value in under two minutes.
